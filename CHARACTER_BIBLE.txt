CHARACTER_BIBLE.txt — LOCKED DESCRIPTORS  (the key to visual consistency)
==============================================================================
WHY THIS EXISTS: AI image tools "forget" what a character looks like between
generations. To keep Karuppar (and everyone else) identical across all 22
sequences, EVERY image prompt must embed the exact LOCKED BLOCK below for each
character that appears. Do not paraphrase these blocks — copy them verbatim.
The reference images live in /Character-Sheets/.

CULTURAL ANCHOR REMINDER:
  Every prompt must also include the GLOBAL_CULTURAL_TOKEN from CONFIG.txt.
  Without it, the model defaults to Western imagery (Vikings, knights, generic
  D&D demons). The LOCKED BLOCKS below already use Tamil-Hindu vocabulary —
  do not "translate" terms like vel, aruval, dhoti, veshti, vibhuti, tilak,
  jata, rudraksha into Western equivalents.

Each block has:
  LOCKED BLOCK         = paste this verbatim into prompts
  NEGATIVES (specific) = per-subject negative overlay, added on top of the
                         global NEGATIVE_PROMPT in CONFIG.txt
  REFERENCE            = which sheet/image to attach as an image reference
                         (--cref / image prompt / IP-Adapter)

render identical, and the model defaults to Western D&D / Tolkien-orc imagery.

SEQUENCE_BACKPATCH_SPEC.txt for the per-sequence variant map). Mix variants in
crowd shots for visual variety.
