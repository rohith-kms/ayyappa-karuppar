Asura Army — Mahishi's Legions  (replaces former /Asuras folder)
================================================================
Five archetypes — expanded from the original 3 variants (Foot-Soldier,
Brute, Commander) to better honor the visual taxonomy the team built:

  Archetype A — Asura Infantry              (5'5"-5'8", expendable horde)
  Archetype B — Beast-Blooded Shock Troop   (6'5"-7'0", animal-headed elite) NEW
  Archetype C — Giant Battlefield Breaker   (10'0"-13'0", was "Brute")
  Archetype D — Asura Chieftain             (7'0"-8'0", was "Commander")
  Archetype E — Mahishi Elite Guard         (8'0"-9'0", her personal guard) NEW

Each archetype has its own LOCKED BLOCK in /CHARACTER_BIBLE.txt §4.

Master plates in this folder:
  Asura-Army_full_formation.png         — full army arrayed in 5 tiers
  Asura-Army_character_system.png       — labeled archetypes side-by-side
                                          with stats — THIS is the canonical
                                          reference

PER-PROMPT WORKFLOW:
1. Check SEQUENCE_BACKPATCH_SPEC.txt for the per-beat archetype mix.
2. Embed the LOCKED BLOCK from CHARACTER_BIBLE.txt §4 for each archetype
   present.
3. Attach Asura-Army_character_system.png in IMAGE REFERENCES.
4. If ≥3 figures of the same archetype, inject the ANTI_CLONING_TOKEN.

ANTI-CLONING (CRITICAL — asuras are the highest-risk archetype):
- Archetype B (Beast-Blooded) MUST vary the beast-head type across figures
  — lion, wolf, hyena, boar-tusked. Never repeat the same beast-head twice
  in adjacent figures.
- Archetype A (Infantry) is the highest-volume rank — vary faces, weapon
  loadout, helm style, build.
- Archetype E (Elite Guard) only appears flanking Mahishi — never alone,
  never in wide horde shots.
